## Mission

## Service Tiers

### Tier 1: Website Security Audit — $399 (Launch: $79)
**Target:** Any business with a website
**Deliverables:**
- External reconnaissance & attack surface mapping
- Security header analysis
- SSL/TLS configuration review
- Technology fingerprinting & known vulnerability check
- OWASP Top 10 web vulnerability scan
- Findings report with prioritized remediation steps
**Timeline:** 1-2 business days

### Tier 2: Small Business Security Assessment — $2,500
**Target:** Businesses with 5-50 employees
**Deliverables:**
- Everything in Tier 1
- Network perimeter assessment
- Email security review (SPF, DKIM, DMARC)
- Cloud configuration review (M365, Google Workspace)
- Password policy & MFA audit
- Employee security awareness evaluation
- Phishing susceptibility assessment (optional add-on)
- Executive summary + full technical report
**Timeline:** 3-5 business days

### Tier 3: Comprehensive Security & Compliance Review — $5,000–$10,000
**Target:** Businesses in regulated industries (healthcare, finance, legal)
**Deliverables:**
- Everything in Tier 2
- Internal network assessment
- Compliance gap analysis (HIPAA, PCI-DSS, NYDFS 23 NYCRR 500, NJ DPPA)
- Vulnerability scanning & validation
- Wireless security assessment
- Incident response plan review (or creation)
- Remediation roadmap with 30/60/90 day plan
- 30-minute executive briefing
**Timeline:** 1-2 weeks

### Ongoing: Monthly Security Retainer — $750–$2,000/mo
**Deliverables:**
- Monthly vulnerability scanning
- Security advisory & patch guidance
- Quarterly phishing simulations
- Incident response on-call
- Annual reassessment included
